# Bundled Sofie manifests

JSON in this folder is loaded on first run and via **Settings → Connection → Reload type manifests from assets**.

Keep these files aligned with [`sofie-demo-blueprints/assets/`](https://github.com/SuperFlyTV/sofie-demo-blueprints/tree/main/assets) when publishing blueprint updates. For SPRÁVY v3 work, **unopus** (this repo) is the source of truth — copy into blueprints after review.

| File | Purpose |
|------|---------|
| `sofie-rundown-editor-piece-types.json` | Piece type definitions and GFX preview templates |
| `sofie-rundown-editor-part-types.json` | Part presets (ILU, SYN, GFX, …) |
| `sofie-rundown-editor-segment-types.json` | Segment presets (Headlines, Opening, …) |
| `spravy-v3-smoke-rundown.json` | End-to-end smoke rundown (`spravy-v3-smoke`) |
